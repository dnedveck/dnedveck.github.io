EditR

Check out the website, the GitHub repo, and the bioRxiv.org preprint.

I got involved with Mitch Kluesner's project in developing an algorithm to use Sanger sequencing to quantify the amount of editing in cells modified with base editing technology. Editing amount is normally quantified using Illumina sequencing, costing much more and taking more time than the cheap (~$3 / sample) and fast (1 day) Sanger sequencing method. I refactored Mitch's algorithm in R, and wrote a Shiny web app to make it more user friendly amd widely available to the community.
